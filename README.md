## 👩🏻‍💻 Installation and setup 

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

The app will usually start on port 3000, to view in browser `localhost:3000`.
